# Ejemplos de Técnicas de Programación en POO

Este repositorio contiene ejemplos en Python de las técnicas de programación
vistos en la asignatura **Programación Orientada a Objetos**:

- Abstracción
- Encapsulación
- Herencia
- Polimorfismo

## Estructura

- `abstraccion/abstraccion_personaje.py`
- `encapsulacion/cuenta_bancaria.py`
- `herencia/herencia_personas.py`
- `polimorfismo/polimorfismo_animales.py`

Cada archivo incluye:
- Código comentado
- Ejemplos de ejecución en la sección `if __name__ == "__main__":`

## Cómo ejecutar

Desde la terminal, ubicándote en la carpeta raíz del proyecto:

```bash
python abstraccion/abstraccion_personaje.py
python encapsulacion/cuenta_bancaria.py
python herencia/herencia_personas.py
python polimorfismo/polimorfismo_animales.py
